# Shea Butter Export Business Analysis

## Overview
This project analyzes global shea butter export markets to identify high-demand countries, pricing trends, and profitability opportunities.

## Key Insights
- Analyzed trade data across 8+ countries
- Identified pricing range of ₹800–₹1200/kg
- Estimated profit margins of ~20–30%
- Identified top 3 target export markets
